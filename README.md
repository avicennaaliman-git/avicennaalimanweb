# Minimalist Interactive Portfolio

A simple, elegant, and interactive portfolio website built with HTML, CSS, and JavaScript.

## Features

- Minimalist design with clean typography and subtle animations
- Responsive layout that works on all devices
- Smooth scrolling navigation
- Fade-in animations on scroll for sections
- Hover effects on project cards

## How to Use

1. Open `index.html` in your web browser.
2. Navigate through the sections using the menu or by scrolling.
3. Enjoy the interactive elements!

## Customization

- Edit `index.html` to change content
- Modify `style.css` for styling adjustments
- Update `script.js` for additional interactivity

## Browser Support

Works in all modern browsers that support ES6 and CSS transitions.
